Primeros pasos HTML

Primer día 11/06/26

Aprendí lo más básico sobre el HTML. Es un lenguaje de marcado que todos los sitios web lo usan. Es básicamente el esqueleto de las páginas web. No tiene color ni tipo de letra, pero tiene la forma general e información básica. Se crea mediante etiquetas. Se usa "<" al inicio de la etiqueta y ">" al final, con una etiqueta como "h1" o "p". Se usa "/" para dar un cierre. "h1" se refiere al título principal de la página, mientras que "h2" al subtítulo y los demás (h3, h4... hasta h6) para más subniveles. El texto principal se escribe adentro de la etiqueta "p", que significa párrafo. CSS es el segundo lenguaje que debería aprender; este da colores y formas a los textos y títulos de las páginas web. El JavaScript es el que da la lógica detrás de las páginas web. Por ejemplo, un botón que te mande a otra página o un mapa interactivo.

Segundo día 12/06/26

Entendí más profundamente la función del HTML (HyperText Markup Language), CSS y JavaScript en una página web. HTML te sirve para mostrar contenido, CSS para darle estilo y JavaScript para dar le el cerebro. En esencia, HTML es el esqueleto que da forma a todo, CSS la carne que le da volumen y JavaScript el cerebro que controla todo.

Imagenes en HTML

Una imagen en HTML se compone de dos atributos, "src" y "alt". Para poner una imagen se usa la etiqueta "img" y luego seguido de "src=" pones de donde sacar la foto y en "alt" describes lo que contiene la imagen, esto ayuda a aplicaciones entender que contiene la imagen.

Un atributo es una palabra especial que se coloca dentro de una etiqueta para dar información adicional o modificar el comportamiento por defecto del elemento.
